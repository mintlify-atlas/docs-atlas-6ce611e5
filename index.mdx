## What is DocuGen AI?

DocuGen AI is a cutting-edge documentation automation tool that leverages Abstract Syntax Tree (AST) analysis and Google Gemini's language models to generate comprehensive, production-ready README files from your Python projects.

Instead of manually writing documentation, DocuGen scans your codebase, extracts structural metadata (classes, functions, methods, type annotations), and uses AI to generate clear, technical documentation that evolves with your code.

## How It Works

<Steps>
  <Step title="Scan & Discover">
    DocuGen scans your Python project, respecting .gitignore rules and identifying all relevant source files
  </Step>
  <Step title="Parse & Extract">
    AST parser analyzes each file to extract classes, functions, methods, parameters, return types, and docstrings
  </Step>
  <Step title="Normalize & Process">
    Metadata is normalized into a clean format optimized for AI comprehension
  </Step>
  <Step title="Generate & Render">
    Google Gemini processes the metadata and generates professional documentation, which is then rendered using Jinja2 templates
  </Step>
</Steps>

## Why DocuGen AI?

<Note>
Traditional documentation quickly becomes outdated as code evolves. DocuGen AI solves this by regenerating documentation on-demand, ensuring your docs always reflect the current state of your codebase.
</Note>

**Benefits:**
- **Save Time**: Eliminate hours of manual documentation writing
- **Stay Accurate**: Generated docs reflect actual code structure
- **Improve Quality**: AI-generated explanations provide clear, professional descriptions
- **Maintain Consistency**: Standardized format across all your projects
- **Focus on Code**: Spend more time building features, less time writing docs
